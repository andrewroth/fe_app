== README

This is an exmple of how to use the Form Engine with Devise.

To set up:

    bundle install
    rake db:setup
    rake fe_engine:db:seed

And run the server.

To fill out the form, just go to the root and use the basic user

To edit the forms, go to /fe/admin/question_sheets as the admin user
